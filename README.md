# DenounceBeats.API

Proyecto Web API desarrollado en ASP.NET Core. 
El repositorio contiene un proyecto ASP.NET Core Web API con un CRUD de la entidad Sector, utilizando Controllers, DTOs, DbContext, configuración en appsettings.json y Entity Framework Core con SQL Server.

## Funcionalidades
- CRUD de Sectors
- Entity Framework Core
- SQL Server
- Swagger

## Endpoints
- GET /api/Sectors
- GET /api/Sectors/{id}
- POST /api/Sectors
- PUT /api/Sectors/{id}
- DELETE /api/Sectors/{id}

## Base de datos
Configurar la cadena de conexión en appsettings.json
